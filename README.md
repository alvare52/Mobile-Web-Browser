# Mobile Web Browser 

Simple iOS web browser using WKWebView framework.

## Features
- Search engine options (Bing, DuckDuckGo, Google, or Yahoo)
- Multiple tabs and basic navigation
- Save bookmarks

## Frameworks
- WKWebView
- UIKit
- URLSession
